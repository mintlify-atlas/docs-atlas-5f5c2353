go-admin is an open-source backend admin framework that combines Gin (HTTP), GORM (ORM), Casbin (RBAC), and JWT authentication into a ready-to-run scaffold. Connect your database, run the migration command, and you have a working admin API with user management, role-based access control, code generation, scheduled jobs, and more — all in under 3 minutes.

## Get started in 3 steps

<Steps>
  <Step title="Clone and configure">
    Clone the repository and edit `config/settings.yml` with your database credentials.
  </Step>
  <Step title="Initialize the database">
    Run `./go-admin migrate -c config/settings.yml` to create all tables and seed data.
  </Step>
  <Step title="Start the server">
    Run `./go-admin server -c config/settings.yml`. The API is live at `http://localhost:8000`.
  </Step>
</Steps>

<Note>
  go-admin requires Go 1.18+. The default configuration uses SQLite so you can try it immediately without installing a separate database server.
</Note>
